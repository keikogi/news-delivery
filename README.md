Keikogi News Delivery Tool
==========================

Requirements
------------
PHP 5.3+

PHP Mongo extension 1.5+

MongoDB 2.6+

Installation
------------
Add this to a composer.json file:
```
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/keikogi/news-delivery"
    }
],
"require": {
    "keikogi/news-delivery": ">=1.0.0"
}
```
